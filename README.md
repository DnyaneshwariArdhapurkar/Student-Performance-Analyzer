# 📊 Student Performance Analyzer (Python)

## 📌 Description
The Student Performance Analyzer is a Python-based application that helps analyze students' marks and evaluate their academic performance.

## ✨ Features
- Calculate total and average marks
- Display performance (Excellent / Good / Average / Poor)

## 🛠️ Tech Stack
- Python 

## ▶️ How to Run
1. Clone the repository  
2. Run the Python file
